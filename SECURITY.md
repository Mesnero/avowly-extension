# Security policy

## Reporting a vulnerability

Please email **security@avowly.io** with details. Do not file public issues for security problems.

We commit to:
- Acknowledge receipt within 72 hours
- Provide an initial assessment within 7 days
- Credit reporters who request it (after fix is deployed)

## Scope

In scope:
- The extension itself (this repository)
- The Chrome Web Store and Firefox AMO listings
- The auto-update path

Out of scope:
- The browser platform itself (report to Chromium / Mozilla)
- Third-party services we depend on
- Social engineering, physical attacks
- Findings that require a malicious extension already installed alongside Avowly

## Coordinated disclosure

We follow a 90-day coordinated-disclosure standard.
